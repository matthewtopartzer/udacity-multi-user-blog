To deploy this app:

1. Download and install Python 2.7 here: https://www.python.org/downloads/ (Google App Engine only uses 2.7)
2. Download and install Google App Engine SDK for Python here: https://cloud.google.com/appengine/downloads#Google_App_Engine_SDK_for_Python
3. Run the Google App Engine
4. Create new application
5. Copy or overwrite the existing default libraries in the same directory
6. Run the application
7. Select Browse or visit localhost:8080 in your browser
